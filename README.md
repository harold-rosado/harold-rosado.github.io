# 🚀 Harold Rosado | UX/UI Designer & Frontend Developer

Welcome to my professional portfolio. This project is an optimized, accessible, and high-performance landing page built with **Astro** and **Tailwind CSS**.

---

## 🛠️ Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** Lucide Icons
- **Forms:** Formspree Integration

## ✨ Features

- 📱 **Mobile First:** Fully responsive design.
- 🎨 **UX/UI:** Focus on accessibility and visual hierarchy.
- ⚡ **Performance:** Ultra-fast loading thanks to Astro's island architecture.
- 📧 **Direct Contact:** Formspree integration for lead management.

## 🧞 Commands

All commands are run from the root of the project:

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Installs the required dependencies          |
| `npm run dev`     | Starts the local server at `localhost:4321` |
| `npm run build`   | Builds the site for production in `./dist/` |
| `npm run preview` | Previews the production build locally       |

## ⚙️ Requirements

- **Node.js:** v24.12.0 (recommended to use `nvm`)
- **Package manager:** npm
